## Did I contaminate this session?

Answer honestly; a session you steered is worse than no session, because it produces
confident wrong conclusions.

- Did I break a silence before ten seconds? **Y** / **N**
- Did I point at anything? **Y** / **N**
- Did I explain what something meant before they asked? **Y** / **N**
- Did I answer a direct question instead of deflecting once? **Y** / **N**
- Did they know I built it? **Y** / **N**

If two or more are **Y**, mark this sheet **suspect** and weight it accordingly when writing
up: ☐ suspect
